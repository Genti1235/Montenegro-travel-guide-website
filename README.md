# Montenegro Travel Guide Website

This repository contains a connected static website built from the travel-guide page set in `stitch_montenegro_wild_beauty_guide.zip`.

## Pages

- `index.html` - main homepage
- `pages/bay-of-kotor.html` - destination guide
- `pages/7-day-road-trip.html` - itinerary page
- `pages/transport.html` - buses and car rentals guide
- `pages/when-to-visit.html` - seasonal advice page
